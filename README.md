# Module 26 – Automation Using Selenium: Getting Data

---

## Overview

This project demonstrates **web scraping with Selenium** in Python.  
It targets **https://books.toscrape.com** — a website built specifically for  
practising web scraping (fully legal, no robots.txt restrictions).

---


---

## What Each Script Does

### `scraper.py` — Multi-page Listing Scraper
- Opens the Books to Scrape homepage using a **headless Chrome** browser
- Iterates through up to **5 catalogue pages** (≈ 100 books)
- From each book card it extracts:
  - **Title**, **Price**, **Star Rating**, **Availability**, **URL**
- Saves all data to `output/books.csv` and `output/books.json`
- Prints a summary with total count and top-rated books

### `detail_scraper.py` — Detail Page Scraper
- Collects book URLs from the listing page
- **Navigates into each book's individual page**
- Extracts richer data not visible in the listing:
  - **Title**, **Price**, **Rating**, **Stock**, **UPC**, **Description**
- Saves to `output/book_details.csv`

## Selenium Concepts Demonstrated

| Concept | Where Used |
|---|---|
| `webdriver.Chrome()` | Launching a controlled browser instance |
| `Options` / `--headless` | Running Chrome without a visible window |
| `driver.get(url)` | Navigating to a URL |
| `WebDriverWait` + `EC` | Waiting for elements to load (explicit waits) |
| `find_element` / `find_elements` | Locating elements by CSS selector |
| `.text` / `.get_attribute()` | Extracting text and HTML attributes |
| Pagination loop | Following "next page" links across multiple pages |
| Multi-page navigation | Visiting individual detail pages per item |
| Exception handling | `NoSuchElementException`, `TimeoutException` |

## Installation

### 1. Install Python dependencies
```bash
pip install -r requirements.txt
```

### 2. Install ChromeDriver
ChromeDriver must match your installed Chrome version.

**Option A – automatic (recommended):**
```bash
pip install webdriver-manager
```
Then in the script, replace:
```python
driver = webdriver.Chrome(options=options)
```
with:
```python
from webdriver_manager.chrome import ChromeDriverManager
driver = webdriver.Chrome(service=Service(ChromeDriverManager().install()), options=options)
```

**Option B – manual:**  
Download from https://chromedriver.chromium.org/downloads and place in PATH.
